## 🖥️ What this app does

stocktrend-ai-forecast is a desktop app for Windows that helps you view stock trend forecasts in a simple way. It uses AI to study market data and show likely price direction, trend shifts, and pattern signals.

This app is made for people who want a clear view of stock movement without digging through charts and data by hand.

## 🔐 API key setup help

The app needs a Gemini API key to run its AI forecast features.

To set it up:

1. Open the app folder.
2. Find `.env.local` if the release includes it.
3. Open the file with Notepad.
4. Add your key in this format:

`GEMINI_API_KEY=your_api_key_here`

5. Save the file.
6. Start the app again.

Keep your API key private and do not share it with others.
